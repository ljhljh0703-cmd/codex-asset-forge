# Notice

This repository contains code, documentation, and generated sample assets used
to explain an AI-assisted game asset pipeline.

## Provenance

- AgentForge sample assets were generated with Codex built-in image generation
  on a solid `#FF00FF` background and postprocessed into keyed PNGs, atlases,
  manifests, and metadata.
- ClaudeCraft R020 tile/item sample images are generated proof artifacts from a
  related local game prototype. They are included as case-study evidence.
- Bone Trail / libGDX review images are review-only evidence for an asset
  governance pipeline. They are not final accepted game art.

## Boundaries

- No Aseprite binary, source, or proprietary editor asset is vendored here.
- No Shattered Pixel Dungeon assets or text are included as project-original
  material.
- Runtime proof is separate from visual or fun acceptance.
- Sample assets are included for demonstration and pipeline study.

## Third-Party Dependencies

The Python toolkit depends on Pillow. See `pyproject.toml`.
