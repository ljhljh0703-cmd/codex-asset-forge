# Examples

These are sample artifacts copied from local proof runs.

## agentforge-starter

Starter atlas pack generated from Codex built-in image generation on a solid
`#FF00FF` background. Includes raw/keyed images, atlas PNGs, JSON metadata,
collision metadata, scene hooks, and QC output.

## claudecraft-r020

Tile/item sprites and proof artifacts from the ClaudeCraft R020 glyph-to-sprite
conversion. Use these as evidence of visible runtime rendering, not as final
visual acceptance.

## bone-trail-voxel-prepass

Review-only voxel turnaround and cleanup candidates from the Bone Trail /
libGDX asset pipeline. These are governance and prepass artifacts, not final
accepted runtime art.
